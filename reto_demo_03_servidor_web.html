Reto Demo 03 Servidor web

Capacidades web

La placa Steamakers tiene capacidades wifi y además puede actuar como servidor o cliente web, todas estas opciones están recogidas en Arduinoblocks. Para complementar estas características presenta opciones para formatear html.

Tarea

Duración:
15:00
Agrupamiento:
1

El reto consiste en que una placa steamakers haga de punto de acceso wifi, y realice funciones de servidor web ofreciendo una web con menú principal y tres opciones:

  • Opción de información: dando datos de temperatura y humedad en una web.
  • Opción de encendido de led RGB (color aleatorio)
  • Opción de apagado de leg RGB

Objetivos:

  • Creación de punto de acceso.
  • Creación y control de servidor web.
  • Edición html.
